# Stocku

## Dependencies
1. 安裝nodemon （讓server 可以在檔案變更之後自動重啟）
```
$ npm install nodemon -g
```

## 安裝:
```
$ git clone https://github.com/0xYTF/stocku
$ cd stocku
$ npm install
```

## 使用:
```
$ npm start
```
or
```
$ nodemon bin/www
```
or
```
$ node bin/www
```

預設的port為 3000
若要更改，請使用
```
$ PORT=???? npm start
```
