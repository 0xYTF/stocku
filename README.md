# Stocku

## Dependencies
1. 安裝nodemon （讓server 可以在檔案變更之後自動重啟）
`npm install nodemon -g`

## 安裝:
`git clone https://github.com/0xYTF/stocku`
<br>
`cd stocku`
<br>
`npm install`

## 使用:
`npm start`
<br>or<br>
`nodemon bin/www`
<br>or<br>
`node bin/www`




預設的port為 3000
若要更改，請使用
<br>
`PORT=???? npm start`
